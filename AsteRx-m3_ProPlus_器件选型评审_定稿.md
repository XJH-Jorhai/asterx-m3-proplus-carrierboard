## 2. 电源架构

最终电源路径：

```text
DC 12V输入/24V误插容错
  -> DC-005-A200 / PTC / TVS / CMC / MOS反压保护
  -> TPS26631 eFuse，当前标注 OVP≈20.8V / UVLO≈10.5V / ILIM≈2.05A
  -> LM76002 同步降压到 5V_DC
  -> TPS2121 5V 电源 MUX
  -> 5V_SYS

5V_SYS 分支
  -> TPS7A8300 LDO
  -> 3V3_ASTERX
  -> TPS7A2042 LDO
  -> 4.2V_ANT
  -> SPX3819 LDO
  -> 3.3V_232485

USB-C VBUS 5V
  -> TPS259472 eFuse/限流/反灌保护，当前标注 OVP≈5.7V / UVLO≈3.8V / ILIM≈2.7A
  -> TPS2121 5V 电源 MUX
  -> 5V_SYS
```

模块 30-pin Pin 1/2 只能接 `3V3_ASTERX`，不能由 `USB_VBUS` 直接供电。30-pin Pin 7 `USB_VBUS` 只作为 USB VBUS 检测。

当前原理图中 `TPS2121RUXR` 标注切换阈值约 `4.75V`，第三级过压阈值约 `5.7V`；`REF3325AIDBZR` 用于相关阈值/比较参考。

## 3. 功耗预算

按板上实际负载估算，`5V_SYS` 主要负载如下：

| 负载项 | 常用折算到 5V_SYS | 设计峰值/预留 | 依据 |
|---|---:|---:|---|
| AsteRx 本体 | 0.56A | 1.00A | 手册全GNSS双天线 + L-Band + WBIM，另按3.3V 1A峰值要求 |
| 双有源天线 VANT | 0.10A | 0.30A | 每路天线限流150mA，MAIN+AUX1两路 |
| Ethernet PHY + RJ45 LED | 0.065A | 0.08A | KSZ8041类PHY <180mW + LED |
| MicroSD | 0.10A | 0.20A | SD卡写入/初始化预留 |
| RS232 | 0.015A | 0.03A | 当前为 RSM232 隔离模块，按样机实测复核 |
| 隔离 RS485 | 0.08A | 0.25A | 当前为 TD301D485H-A 集成隔离模块 + 总线负载 |
| Event/PPS/REF逻辑 | 0.01A | 0.03A | OR逻辑/整形/ESD预留 |
| 面板LED/状态灯 | 0.03A | 0.05A | LOGLED/GPLED/POWER LED |
| 电源路径芯片自身 | 0.01A | 0.02A | eFuse/MUX/USB保护静态电流 |
| 合计 | 0.97A | 1.96A | 不含额外对外5V供电 |

加 15% 工程余量后，`5V_SYS` 设计目标约 `2.25A`。`LM76002` 为 2.5A 同步 buck，可以覆盖当前底板，但余量不宽。原理图和 PCB 按 `LM76002/LM76003` 兼容设计，如果实测峰值偏高，可不改板切换到 `LM76003`。

限制条件：

- 不增加 USB Host / Type-A。
- 不对外提供额外大电流 5V。
- RS232/RS485 由 `3.3V_232485` 输入侧电源供电，隔离侧由所选隔离模块内部提供；不再按外置 `B0505S-1WR3` 作为主方案。
- AsteRx 3.3V LDO 不长期按 2A 输出使用。
- 双天线 VANT 必须限流。

## 4. 关键器件说明

### 4.1 TPS26631

`TPS26631RGER` 是 DC 入口保护主选。此前评审中的 `TPS26633RGER` 不再作为主选，原因是本项目需要在 24V 误插时切断 DC 输入，而 `TPS26631` 更符合当前 UVLO/OVP/ILIM 可编程保护边界。

设计注意：

- 设定 OVP、UVLO、限流、软启动。
- 前端 TVS、保险丝或保护电阻与 eFuse 参数配合。
- `TPS26633RGER` 不作为本项目采购备选；如确需替代，必须重新确认是否仍能满足 24V 过压切断需求。
- 如库存异常，可评估同系列其他带外部 OVP 的器件或 `TPS26600PWPR`，但后者只有 2A 级，不应直接替代而不重算功耗。

当前原理图标注配置目标如下：

| 项目 | 当前目标 |
|---|---:|
| UVLO | 约 `10.5V` |
| OVP | 约 `20.8V` |
| ILIM | 约 `2.05A` |

说明：该 OVP 目标会在 24V 误插时提前切断输入。`TPS26631` 的限流作用在 DC 输入侧，不能直接沿用 `5V_SYS` 的输出侧电流；量产前必须按已放置分压电阻、`R_ILIM` 和 TI 数据手册公式重新复算阈值。`UVLO≈10.5V` 对 12V 标称输入的跌落裕量有限，需用实际适配器和线缆压降验证。

### 4.2 LM76002 / LM76003

`LM76002RNPT` 为当前主选，原因是库存更合适且功耗预算可覆盖。电气上 `LM76003` 余量更大，因此 PCB 应保持兼容。

设计注意：

- 使用同一 WQFN-30-EP(4x6) footprint。
- 电感、电容、热设计按更保守的 `LM76003` 余量处理。
- `PGOOD` 引到后级 EN 或测试点。
- 预留 snubber 和输入 EMI 滤波位置。
- SW 节点面积小，输入高 di/dt 回路短，FB 远离 SW。

### 4.3 TPS7A8300

`TPS7A8300RGRR` 为 `3V3_ASTERX` 主 LDO。它的 2A 能力是余量，不代表允许 2A 长期输出。

热耗估算：

```text
典型满功能: (5.0V - 3.3V) * 0.56A ≈ 0.95W
峰值1A:     (5.0V - 3.3V) * 1.00A = 1.7W
```

PCB 必须大面积铺铜、热过孔，并在样机上测温。

### 4.4 RJ45

`C54408 / HR911130A` 为主选。它是千兆带灯带磁 RJ45，但 AsteRx 以太网实际为 10/100，千兆 RJ45 不会提升带宽。选择它的理由是库存和通用性。

原理图阶段核对：

- 10/100 只使用两对线。
- 未用线对按数据手册处理。
- 中心抽头、LED极性、屏蔽壳接地和 PHY 推荐电路一致。

### 4.5 MicroSD

主选 `MOLEX 5025700893`，Push-Push 自弹。需要确认卡检测脚、插拔方向、外壳开口和弹出空间。

### 4.6 30P/60P 板对板

严格遵守 Septentrio 手册，不做自由替代：

- 30P 底板侧：`HRS DF40C-30DP-0.4V(51)`。
- 60P 底板侧：`HRS DF40HC(3.5)-60DS-0.4V(51)`。

DP/DS、公母、0.4mm pitch、3.5mm 堆叠高度、Pin1方向、keepout 任一项错误都会导致装配失败或损坏模块。

### 4.7 DC 电源口

当前原理图放置 `DC-005-A200 / C720557`。该器件与此前文档中的 `5525` 方向不完全一致，机械外壳、适配器插头和额定电流需在下单前单独复核。丝印建议仍按系统功能表达：

```text
DC 12V IN
Center Positive
24V tolerant
```

若最终仍要求 5.5x2.5mm/5525 机械规格，应替换当前 DC 座料号和封装，不能只改 BOM。

### 4.8 接口板 FPC

接口板和底板连接使用 12P 0.5mm FPC。FPC 只承载低速小电流信号，如 reset、nPDN、LOG Button、LOGLED、GPLED、电源开关 EN/LED 等，不承载主电流。

需要确认：

- 同向/反向 FFC。
- 触点方向。
- 加强板厚度。
- 弯折半径和固定方式。

### 4.9 隔离串口与高密度 D-Sub

当前 `232/485串口` 图页按集成隔离模块实现：

- COM1：`TX1/RX1 -> RSM232 -> RS232_TXD_DB9/RS232_RXD_DB9`，经自恢复保险、TVS 和 GDT 后接 `DSUB1` 下组 pin2/pin3，pin5 为 `GND_RS232_ISO`。
- COM2：`TX2/RX2 -> TD301D485H-A -> RS485_A/RS485_B`，不用 `RTS2/DE/RE`，A/B 经 `SMBJ12CA`、共模电感和 `SM712-TP` 后接 `DSUB1` pin10/pin11，pin14 为 `GND_RS485_ISO`。
- `DSUB1` 为 `HYCW195-VGA18B-195B / C42421589` 高密度 D-Sub 母 18P，固定脚/壳体 pin19-22 接 `PGND`。
- `TM-03R` 三位拨码控制 `120Ω` 终端电阻和两只 `680Ω` 偏置电阻接入。
- `GND_RS232_ISO`、`GND_RS485_ISO` 与 `PGND`/D-Sub 壳体之间只允许通过已定义的保护件网络关联，PCB 收尾时需查网确认没有无意硬短。

## 5. 原理图阶段待闭环

1. `TPS26631` 的 OVP、UVLO、ILIM、dVdt 参数；当前原理图标注为 `OVP≈20.8V, UVLO≈10.5V, ILIM≈2.05A`，需按最终电阻复算。
2. `LM76002/003` 的电感、电容、频率、软启动、PGOOD 连接。
3. `TPS7A8300` 热设计和 3V3_ASTERX 实测温升。
4. RJ45 `HR911130A` pinout 与 PHY 参考设计。
5. 60P `DF40HC(3.5)-60DS-0.4V(51)` 采购和实物公母确认。
6. `DC-005-A200 / C720557` 与外壳开孔、适配器插头、额定电流和丝印的一致性。
7. SMA/USB/DC 接口区作为唯一 `CHASSIS-PGND` 硬连接区域；确认 PPS/REF SMA shell、USB Shield、DC shell、`DC-` 过 CMC 前后连接关系，且 Buck/RF/TNC 区不新增硬连接。
8. `4.2V_ANT` 与 AsteRx `VANT` 网名/连接关系。
9. `RSM232`、`TD301D485H-A` 的隔离耐压、波特率、自动方向时序和 D-Sub pinout。
10. USB `OVP≈5.7V` 和 DC `UVLO≈10.5V` 的实际裕量，防止正常 USB 过冲或 12V 输入跌落造成误触发。
