## 2. 电源架构

最终电源路径：

```text
DC 12V输入/24V误插容错
  -> TPS26633 eFuse / TVS / 输入保护
  -> LM76002 同步降压到 5V_DC
  -> 5V 电源 MUX
  -> 5V_SYS
  -> TPS7A8300 LDO
  -> 3V3_ASTERX

USB-C VBUS 5V
  -> TPS259470 eFuse/限流/反灌保护
  -> 5V 电源 MUX
  -> 5V_SYS
```

模块 30-pin Pin 1/2 只能接 `3V3_ASTERX`，不能由 `USB_VBUS` 直接供电。30-pin Pin 7 `USB_VBUS` 只作为 USB VBUS 检测。

## 3. 功耗预算

按板上实际负载估算，`5V_SYS` 主要负载如下：

| 负载项 | 常用折算到 5V_SYS | 设计峰值/预留 | 依据 |
|---|---:|---:|---|
| AsteRx 本体 | 0.56A | 1.00A | 手册全GNSS双天线 + L-Band + WBIM，另按3.3V 1A峰值要求 |
| 双有源天线 VANT | 0.10A | 0.30A | 每路天线限流150mA，MAIN+AUX1两路 |
| Ethernet PHY + RJ45 LED | 0.065A | 0.08A | KSZ8041类PHY <180mW + LED |
| MicroSD | 0.10A | 0.20A | SD卡写入/初始化预留 |
| RS232 | 0.015A | 0.03A | TRS3232E类收发器 |
| 隔离 RS485 | 0.08A | 0.25A | B0505S-1WR3 1W隔离电源 + 总线负载 |
| Event/PPS/REF逻辑 | 0.01A | 0.03A | OR逻辑/整形/ESD预留 |
| 面板LED/状态灯 | 0.03A | 0.05A | LOGLED/GPLED/POWER LED |
| 电源路径芯片自身 | 0.01A | 0.02A | eFuse/MUX/USB保护静态电流 |
| 合计 | 0.97A | 1.96A | 不含额外对外5V供电 |

加 15% 工程余量后，`5V_SYS` 设计目标约 `2.25A`。`LM76002` 为 2.5A 同步 buck，可以覆盖当前底板，但余量不宽。原理图和 PCB 按 `LM76002/LM76003` 兼容设计，如果实测峰值偏高，可不改板切换到 `LM76003`。

限制条件：

- 不增加 USB Host / Type-A。
- 不对外提供额外大电流 5V。
- RS485 隔离电源按 1W 级设计。
- AsteRx 3.3V LDO 不长期按 2A 输出使用。
- 双天线 VANT 必须限流。

## 4. 关键器件说明

### 4.1 TPS26633

`TPS26633RGER` 是 DC 入口保护主选。它不是采购意义上的唯一器件，但在 TI TPS2663 系列里功能最完整，适合本项目的 12V 正常输入、24V 误插容错和工业入口保护。

设计注意：

- 设定 OVP、UVLO、限流、软启动。
- 前端 TVS、保险丝或保护电阻与 eFuse 参数配合。
- 如库存异常，可评估 `TPS26631RGER` 或 `TPS26600PWPR`，但后者只有 2A 级，不应直接替代而不重算功耗。

### 4.2 LM76002 / LM76003

`LM76002RNPT` 为当前主选，原因是库存更合适且功耗预算可覆盖。电气上 `LM76003` 余量更大，因此 PCB 应保持兼容。

设计注意：

- 使用同一 WQFN-30-EP(4x6) footprint。
- 电感、电容、热设计按更保守的 `LM76003` 余量处理。
- `PGOOD` 引到后级 EN 或测试点。
- 预留 snubber 和输入 EMI 滤波位置。
- SW 节点面积小，输入高 di/dt 回路短，FB 远离 SW。

### 4.3 TPS7A8300

`TPS7A8300RGRR` 为 `3V3_ASTERX` 主 LDO。它的 2A 能力是余量，不代表允许 2A 长期输出。

热耗估算：

```text
典型满功能: (5.0V - 3.3V) * 0.56A ≈ 0.95W
峰值1A:     (5.0V - 3.3V) * 1.00A = 1.7W
```

PCB 必须大面积铺铜、热过孔，并在样机上测温。

### 4.4 RJ45

`C54408 / HR911130A` 为主选。它是千兆带灯带磁 RJ45，但 AsteRx 以太网实际为 10/100，千兆 RJ45 不会提升带宽。选择它的理由是库存和通用性。

原理图阶段核对：

- 10/100 只使用两对线。
- 未用线对按数据手册处理。
- 中心抽头、LED极性、屏蔽壳接地和 PHY 推荐电路一致。

### 4.5 MicroSD

主选 `MOLEX 5025700893`，Push-Push 自弹。需要确认卡检测脚、插拔方向、外壳开口和弹出空间。

### 4.6 30P/60P 板对板

严格遵守 Septentrio 手册，不做自由替代：

- 30P 底板侧：`HRS DF40C-30DP-0.4V(51)`。
- 60P 底板侧：`HRS DF40HC(3.5)-60DS-0.4V(51)`。

DP/DS、公母、0.4mm pitch、3.5mm 堆叠高度、Pin1方向、keepout 任一项错误都会导致装配失败或损坏模块。

### 4.7 DC 电源口

按 `5525` 冻结：5.5x2.5mm，中心正极。丝印建议：

```text
DC 12V IN
5.5x2.5 Center Positive
24V tolerant
```

不依赖 5521/5525 松配合兼容。若需要兼容 5521，使用转接线或另设端子方案。

### 4.8 接口板 FPC

接口板和底板连接使用 12P 0.5mm FPC。FPC 只承载低速小电流信号，如 reset、nPDN、LOG Button、LOGLED、GPLED、电源开关 EN/LED 等，不承载主电流。

需要确认：

- 同向/反向 FFC。
- 触点方向。
- 加强板厚度。
- 弯折半径和固定方式。
