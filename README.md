# AsteRx-m3 ProPlus Carrierboard

This repository contains the design boundary, component selection review, and implementation notes for an AsteRx-m3 ProPlus OEM receiver carrier board.

## Contents

- `AsteRx-m3_ProPlus_底板设计规范.md` - carrier board design specification.
- `AsteRx-m3_ProPlus_器件选型评审_定稿.md` - final component selection review.
- `AsteRx-m3_ProPlus_器件选型清单_定稿.csv` - final component selection table.
- `asterx_m3_proplus_carrier_feature_boundary.csv` - feature boundary and design scope table.

## Current Hardware Direction

- AsteRx-m3 ProPlus OEM module on a custom carrier board.
- USB-C device port with optional 5V power input.
- DC 12V input, 24V tolerant front end.
- 5V system rail generated by synchronous buck.
- 3.3V low-noise rail for AsteRx module.
- Ethernet through RMII PHY and RJ45 MagJack.
- MicroSD logging socket.
- COM1 RS232, COM2 isolated RS485, COM3/COM4 LVTTL.
- MAIN and AUX1 antennas via MMCX-to-SMA pigtails.
- PPS, Event, REF IN/OUT external timing interfaces.
- Separate button/interface board connected by FPC.

Vendor manuals are intentionally not committed to this public repository. Refer to the official Septentrio documentation for authoritative module specifications.
